Bubble Sort

Bubble sort, sometimes referred to as sinking sort, is a simple sorting algorithm that repeatedly steps through the input list element by element, comparing the current element with the one after it, swapping their values if needed. This simple algorithm performs poorly in real world use and is used primarily as an educational tool. Average Performance: O(n^2)
