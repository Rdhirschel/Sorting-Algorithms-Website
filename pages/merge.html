Merge Sort

merge sort (also commonly spelled as mergesort) is an efficient, general-purpose, and comparison-based sorting algorithm. Most implementations produce a stable sort, which means that the relative order of equal elements is the same in the input and output. Conceptually, a merge sort works as follows: divide the unsorted list into n sublists, each containing one element (a list of one element is considered sorted). Afterwards, Repeatedly merge sublists to produce new sorted sublists until there is only one sublist remaining. This will be the sorted list. Average Performance: O(nlog(n))
