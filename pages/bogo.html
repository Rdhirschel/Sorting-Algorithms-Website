Bogo Sort

The following is a description of the bogo sort algorithm in pseudocode:

while not sorted(deck):
shuffle(deck)

So basically... Best case Scenario is O(1) but worse case on large datasets would literally be larger than the heat death of the universe (Average Performance: O(n*n!))
